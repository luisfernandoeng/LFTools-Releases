# LFTools

LFTools e um add-in para Autodesk Revit criado para acelerar tarefas de modelagem, documentacao, eletrica, automacao e limpeza de projetos BIM.

Este repositorio publico concentra apenas os arquivos de distribuicao do plugin:

- Instaladores MSI publicados em Releases.
- Arquivo `config.json` usado pelo LFTools para avisos, atualizacoes e controle remoto de ferramentas.

O codigo-fonte do plugin e mantido em repositorio privado.

## Download

Baixe sempre a versao mais recente pela pagina:

[Baixar ultima versao](https://github.com/luisfernandoeng/LFTools-Releases/releases/latest)

Na pagina da release, escolha o instalador MSI adequado:

- `SingleUser`: instalacao por usuario, normalmente sem permissao de administrador.
- `MultiUser`: instalacao para todos os usuarios da maquina, podendo exigir permissao de administrador.

## Instalacao

1. Feche o Revit antes de instalar ou atualizar.
2. Baixe o arquivo MSI da ultima release.
3. Execute o instalador.
4. Abra o Revit novamente.
5. A aba `LF Tools` aparecera na faixa de opcoes do Revit.

## Atualizacoes

Quando uma nova versao estiver disponivel, o LFTools podera avisar automaticamente dentro do Revit e abrir a pagina da ultima release.

Link usado pelo plugin para consultar configuracoes:

```text
https://raw.githubusercontent.com/luisfernandoeng/LFTools-Releases/main/config.json
```

## Requisitos

- Autodesk Revit compativel com a versao instalada do LFTools.
- Windows.
- Conexao com a internet para checar atualizacoes e configuracoes remotas.
